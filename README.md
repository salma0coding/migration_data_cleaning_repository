# Migration Data Cleaning & Transformation (Python - Pandas)

 Project Overview:
This project focuses on cleaning and transforming a real-world migration dataset using Python (Pandas). The goal was to prepare raw, inconsistent data into a clean and structured format ready for analysis.

---

 Tasks Performed:

Data Cleaning:
- Removed extra spaces from column names and text values
- Handled missing values and analyzed data quality
- Removed duplicate records

 Data Transformation:
- Split multi-value columns (Region & Country)
- Matched and expanded rows correctly
- Standardized text formatting

 Data Processing:
- Filtered data based on specific conditions (years, regions)
- Classified African regions into categories
- Created region codes and labels

 Advanced Logic:
- Applied proportional redistribution of population data (males, females, children)
- Validated totals to ensure data consistency
- Removed duplicate IDs for clean dataset structure
